static

Implements the Edmonds-Karp algorithm for finding maximum flow in a graph.

This algorithm finds the maximum flow from a source to a sink in the graph. It uses breadth-first search to find augmenting paths and then augments flow along those paths.

Parameters
gThe graph.
sourceThe optional source vertex code. If empty, the super source vertex is used.
targetThe optional target vertex code. If empty, the super sink vertex is used. @Complexity O(V * E^2) @Complexity V = number of vertexes @Complexity E = number of edges
